# Library Management System

## Overview
Library Management System is a software project developed to automate and manage library operations digitally. It helps in managing books, students, issue and return records efficiently.

## Features
- Add, update, and delete book records
- Manage student/user details
- Track available books
- Maintain library records
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
